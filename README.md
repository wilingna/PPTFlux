🧠 核心理念

❗不是“做 PPT”，而是“重构表达系统”

PPT-FLUX Studio =

AI 资料理解能力（NotebookLM 思维）

⚙️ 工作流（4 Agent 串联）
资料 / 主题
    ↓
Agent 1 · 资料理解（Claude · NotebookLM式梳理）
    ↓
Agent 2 · 结构架构（逻辑 / 页数 / 布局规划）
    ↓
Agent 3 · 表达升级（类 Gemini 的高质量表达）
    ↓
Agent 4 · HTML Slides 渲染（交互式前端呈现）
    ↓
可编辑 HTML 文件（下载 / 修改 / 演示）
